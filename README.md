
# Docket
A web dashboard for managing your course deadlines.

## FAQ

## 1.How to clone the project?
````git clone https://github.com/0sf/docket````

## 2. How we can setup the project?
 ## * First you want install xammp.
   https://www.apachefriends.org/index.html

## * Then you want install composer(Dependency installer)
https://getcomposer.org/Composer-Setup.exe

## * Then you goto the  project directory via cmd or visual studio enter those commands
   1. Install the dependecies
   
     composer global install
   2. Install the fortify library
   
     composer require laravel/fortify 
   3. Install the  jetstream library
 
     composer require laravel/jetstream
   4. Update depencies
   
     composer update
     
   
## 3. How we can preview the project
 1. start the xampp apache server and mysql server.
 2. Go to the prject directory and enter this artisan command
  
        php artisan serve 
   
 3. Then go to the given link
 http://127.0.0.1:8000
